# Whiteboard

Whiteboard is a web app allowing for better organization and planning of homework assignments

## Installation

### Requirements

#### Javascript
- axios
- ical
- @json2csv/plainjs
- express

#### Python
- icecream
- flask
- tabulate
- ics
